# YOUTUBE CLONE APP 🎬

> 유튜브 클론 코딩 연습

[AZTube 데모 링크](https://aztubes.herokuapp.com/)

## Skills

- React.js
- Node.js
- MongoDB
- Redux

## Functions

- 유튜브 API를 이용한 검색 기능과, iframe 태그를 사용한 플레이어 삽입
- 직접 업로드한 비디오 리스트, 구독한 채널의 영상 리스트
- 비디오 업로드 구현 (React DropZone, multer, ffmpeg 사용)
- 채널 구독하기, 비디오 좋아요/싫어요 기능 구현
- 코멘트 기능 구현
- Redux을 이용한 유저 인증 로직을 위한 HOC 구현 (페이지 접근방지, 허용)
- 회원가입, 로그인, 비디오 업로드, 구독 추가 삭제, 리스트 API구현 (node.js + Mongoose)
